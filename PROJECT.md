# Coding Rules

- One feature at a time.
- Every feature must compile before moving forward.
- No duplicate logic.
- Shared code goes inside core/.
- Features remain isolated.
- Every completed feature is frozen.
- Git commit after every completed feature.
- Git push after every completed feature.
